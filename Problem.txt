Тестовое задание JS

Необходимо создать простенькое single-page приложение из 4х страниц. При этом использовать архитектурный паттерн MVC/MVVM как удобно.
С фрэймворками можно не заморачиваться а использовать JS в чистом виде.
Итак нужно реализовать:
1. Роутер
2. View
3. Model
4. Controller (для MVC)
5. Шаблонизатор (можно использовать любой существующий, можно придумать простенький свой). Обратите внимание, что хедер футер должен быть переиспользован.
6. Взаимодействие с сервером (данные берутся из файла friends.json) Простенький ajax (либо любая готовая реализация в MVC/MVVM фрэйворках).

Страницы:
1. About – текстовая страница с какой-то информацией
2. Search – страница для поиска друзей. После ввода первых двух букв осуществляем поиск по имени. Выводим результат поиска на страницу.
3. Friends – просто страница со списком друзей из файла friends.json
4. Detail – на эту страницу попадаем по клику по имени со страницу поиска или Friends. Содержит все данные о записи.

Допускается использование любых удобных фрэймворков либо же их отсутствие (чистый JS – наиболее интересный вариант).
На тестовое задание можно выделить от 4х часов до 2х дней.
Оцениваться будет в осном архитектурное решение и возможность мастабировать проект.
